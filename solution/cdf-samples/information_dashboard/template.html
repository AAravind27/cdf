Community Dashboard Framework - Home

This framework provides a standardized way to implement and publish dashboards to the Pentaho BI Platform. The Pentaho AJAX API is used to retrieve the BI content and the Pentaho Solution Repository stores the content definitions. Therefore all dashboard information objects, such as charts and reports, can be defined in Pentaho Action Sequences.
